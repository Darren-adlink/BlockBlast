# 🎮 Block Blast Solver - AI 智能求解器

使用先進的位元運算技術，快速破解 Block Blast 遊戲難題！

## ✨ 特色功能

- 🧠 **AI 自動求解**：深度優先搜索 + BigInt 位元運算，快速找出最佳解
- 🎯 **手動模擬模式**：在沙盒環境中測試你的策略
- 🎨 **視覺化演示**：逐步展示 AI 的解題過程
- ⚡ **高效能運算**：位元矩陣運算，處理速度極快

## 🚀 快速開始

### 安裝依賴
\`\`\`bash
npm install
\`\`\`

### 開發模式
\`\`\`bash
npm run dev
\`\`\`
訪問 http://localhost:5173 查看應用

### 打包部署
\`\`\`bash
npm run build
\`\`\`

## 📖 使用說明

### 編輯模式
1. 點擊盤面格子設定初始狀態（藍色 = 已佔用）
2. 點擊手牌區格子繪製方塊形狀
3. 可使用清空按鈕重置盤面或方塊

### AI 求解
1. 點擊「AI 計算最佳解」
2. AI 會嘗試所有可能的排列組合
3. 找到解答後會進入演示模式
4. 使用「上一步」「下一步」查看解題過程

### 手動模擬
1. 點擊「手動模擬」進入沙盒模式
2. 選擇手牌中的方塊
3. 點擊盤面放置
4. 自動消除滿行/滿列
5. 可使用「上一步」撤銷操作

- **Lucide React** - 圖示庫

## 🧮 核心算法

### 位元矩陣運算
- 使用 BigInt 表示 8x8 盤面狀態
- 位元操作實現快速放置與消除判定
- 時間複雜度：O(1) 單次操作

### DFS 求解器
- 深度優先搜索所有可能的放置順序
- 包含排列組合優化
- 支援剪枝加速
